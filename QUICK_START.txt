STEP 1: Verify Changes

$ python3 -m py_compile free-claude-code/api/dependencies.py
$ python3 -m py_compile free-claude-code/api/app.py

$ python3 -m py_compile free-claude-code/api/models/anthropic.py
✅ All should compile without errors

STEP 2: Revoke Exposed API Keys
────────────────────────────────
1. NVIDIA Build: https://build.nvidia.com/settings/api-keys
2. OpenRouter: https://openrouter.ai/keys
3. Telegram: Message @BotFather /token
4. Discord: Developer Portal → Regenerate Token

STEP 3: Create Production Config
─────────────────────────────────
$ cp free-claude-code/.env.production.example .env.production
$ nano .env.production
Set:
  - ENVIRONMENT=production
  - ANTHROPIC_AUTH_TOKEN=<secure-random-token-32-chars>
  - NVIDIA_NIM_API_KEY=<new-key>
  - OPENROUTER_API_KEY=<new-key>
  - Other tokens...

STEP 4: Test Locally
────────────────────
$ source .env.production
$ cd free-claude-code
$ uvicorn api.app:app --host 127.0.0.1 --port 8082

In another terminal:
$ curl http://localhost:8082/  # Should return 401 (no auth)
$ curl -H "x-api-key: $(echo $ANTHROPIC_AUTH_TOKEN)" http://localhost:8082/  # Should work
$ curl -I http://localhost:8082/ | grep X-  # Should show security headers

STEP 5: Deploy to Production
──────────────────────────────
Set environment variables and restart service:
  ENVIRONMENT=production
  ANTHROPIC_AUTH_TOKEN=<token>
  (all other API keys)

Verify: curl https://your-production-server/ \
  -H "x-api-key: <token>"
